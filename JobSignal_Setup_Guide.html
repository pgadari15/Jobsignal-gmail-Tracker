JobSignal Setup Guide

Gmail → Google Sheets → Auto Dashboard · ~10 minutes to set up
🤖 What this agent does for you
📧
Reads your GmailScans for application confirmations, rejections, interview invites
🧠
Classifies automaticallyApplied, Advancing, Interview, Rejected, Offer — no manual tagging
📊
Updates your dashboardLive stats: total, response rate, high priority actions
⏰
Runs every morningAuto-syncs at 8 AM daily — always up to date
💡
AI action suggestionsTells you what to do next for each application
🔒
100% privateData stays in YOUR Google account — nothing sent externally
1
Create a new Google Sheet
Open a new blank spreadsheet. Name it "JobSignal Tracker" so you can find it easily.
→ Open Google Sheets
2
Open Apps Script editor
In your new sheet, click the menu bar:

Extensions → Apps Script

A new tab will open with a code editor. This is where the agent lives.
💡 If you see existing code (like "function myFunction()"), delete all of it first.
3
Paste the agent code
Open the file JobTracker_AppsScript.js (downloaded with this guide). Select all the code (Ctrl+A / Cmd+A), copy it, and paste it into the Apps Script editor.
Ctrl+A to select all → Ctrl+C to copy → paste into Apps Script editor
💡 Then click the Save icon (💾) or press Ctrl+S
4
Run: Setup Sheets
In the Apps Script editor, click the function dropdown (top center) and select setupSheets. Then click the ▶ Run button.

Google will ask for permissions — click Allow (it needs Gmail + Sheets access to work).
⚠️ You may see "Google hasn't verified this app" — click "Advanced" → "Go to JobSignal (unsafe)" — this is YOUR script in YOUR account, it's safe.
5
Run: Sync Gmail Now
Change the function dropdown to syncGmailJobs and click ▶ Run.

The agent will scan up to 200 emails from the past year. This may take 30–60 seconds. A popup will confirm when done!
⏱ Expected time: 30–90 seconds for first sync
6
Enable Daily Auto-Sync
Run the function createDailyTrigger once. After that, the agent will automatically scan your Gmail every morning at 8 AM and update your dashboard — no action needed from you.
💡 Or use the custom menu in your Sheet: "🤖 JobSignal Agent" (appears after first run)
7
View your Dashboard!
Go back to your Google Sheet. You'll see 3 tabs:

📊 Dashboard — your live KPIs (total, response rate, high priority actions)
📋 Applications — every email classified with AI-suggested action
🔍 Agent Log — full history of every sync the agent ran
💡 To add an application manually, use the "🤖 JobSignal Agent" menu → Add Entry Manually
